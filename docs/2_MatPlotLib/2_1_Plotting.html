Plotting Data#

Steal the code below to plot data in a simple x,y plot. Note that we set x and y to be two different lists of values. You can set the x and y to any two sets of data that have the same number of entries. One list might be rates and another might be concentrations of reactants, for example. Consider the code below.

### Set up the data lists

conc = [1.1, 2.0, 2.8, 3.5, 5.0, 7.5, 9.8]     ### units are mM
rates = [18.7, 28.6, 35.0,  39.4, 46.5, 54.1, 58.5]   ### units are uM/min
Show code cell source Hide code cell source 
from matplotlib import pyplot as plt       ### load in the plotting tools from the matplotlib package

### set x and y so that we never need to change the plotting code below

x = conc
y = rates

### A basic plot of x,y data

plt.plot(x,y)

the PyPlot sublibrary of the MatPlotLib package has many commands that will enable us to control the style and features of our plot. Consider the code below.

### Create blank plot

plt.figure(figsize=(4,3))    ### sets the size of the plot figure in inches

More Examples of Formatting#

The PyPlot sublibrary has access to many many features and controls for plotting. Consider the example below.

plt.figure(figsize=(4,3))    ### sets the size of the plot figure in inches

### Plot the data with style
plt.plot(x, y,

Plotting Functions#

We will plot an exponential decay curve. We will plot y = e–x. First we need a x-axis. As you see below, we will use the linspace tool from NumPy to create an evenly spaced array of values between two points. We will then calculate a new array for the y-axis by applying the exponential decay function. We now have a set of x,y values to plot.

We will then plot the x,y data using tools from the plt object (which is actually the matplotlib.pyplot library.) Now we will have our plot.

import numpy as np

x = np.linspace(0,5,100)  # make an array of 100 evenly paced values beltween 0 and 5.
y = np.exp(-x)            # make an array of calculated values

plt.plot(x,y)             # plot y vs. x
plt.show()                # show the plot

The Same Thing, Only Better#

We can add elements to the plot such as titles and labels. We can set limits on the x- and y-axis. We can change the color of the line. Examine the code below and see how we did all that.

The commands executed below are called “methods”. They are functions built into the plt object. They modify the object according to the input for the method. When you enter plt.ylabel("Amount"), the plt object has a label added to its y-axis. In the last line, the plt.show() method displays and clears the plt object. Entering plt.show() a second time will display nothing unless you build a new plot.

There are much more sophisticated ways to present plots and we will explore some aspects of this later. However, this is all the style that we need to know for now.

import numpy as np                 # A library of math functions and data structure
import matplotlib.pyplot as plt    # A library of plotting functions

x = np.linspace(0,5,100)           # make an array of 100 evenly paced values beltween 0 and 5.
y = np.exp(-x)                     # make an array of calculated values

plt.plot(x,y, "k--")               # plot y vs. x: the "k" means black, "--" means dashed line

plt.title("Exponential Decay")     # Title of the plot         
plt.xlabel("Time")                 # x-axis label   
plt.ylabel("Amount")               # y-axis label  

plt.xlim([-1,5])                   # set x-axis range
plt.ylim([0,1.2])                  # set y-axis range

plt.show()                         # show the plot

Two Plots in a Figure#

Let us demonstrate first order reaction kinetics. We will plot the concentration of reactant and product for the reaction A → B. We know that the instantaneous rate at any given time point is…

\[-\frac{\partial [A]}{\partial t}=\frac{\partial [B]}{\partial t}=k[A] \]

…and the integrated rate equation that describes the concentrations of A and B as they change over time are

\[ [A]_t=[A]_0 e^{-k t} \text{ and } [B]_t=[A]_0 \left( 1-e^{-k t} \right)\]

Below we will create an array of values for the x-axis and then make two calculated arrays for the y-axis. One set is for [A] and the other is for [B]. We will then plot them.

A0 = 1.5                  # create a variable that represents the initial concentration
k = 0.34                  # a variable that represents the rate constant

t = np.linspace(0,5,100)  # make an array of 100 evenly paced values beltween 0 and 5 for time
yA = A0*np.exp(-k*t)      # make an array of calculated values for [A]
yB = A0*(1-np.exp(-k*t))  # make an array of calculated values for [B]

plt.plot(x,yA)            # plot y vs. x
plt.plot(x,yB)            # plot y vs. x

plt.show()                # show the plot

Summary#

We have learned how to make a set of values for an x-axis, calculate a corresponding set of values for the y-axis and plot the data. We have seen that repeated plot commands will add to the plot until a plt.show() function is called to display the results.

There are many, many options and features to the MatPlotLib plotting system. We have learned how to add titles and labels and to set the color and style of the line. We can have multiple plots in a single figure, we can save the plots in a variety of formats and so much more. The NumPy tools and other toolsets that we import will provide many useful functions for math and interpreting data. Keep an eye out for new opportunities to learn more about styling plots. We will learn as we go.
